# Quest Craft GM AI Assistant Prototype

A prototype AI assistant for Quest Craft Game Masters. It takes a current table situation as input and generates 2–3 possible story options, which can be regenerated, revised, or developed further into a full narration with consequences, next steps, and age/safety notes.

## How It Works
1. **Input** — Enter your current table situation (or load the demo scenario).
2. **Generate Suggestions** — The assistant (a modified Google Gemini 3.5 Flash model) returns 2–3 possible story options.
3. **Regenerate / Revise** — Regenerate all options or submit specific revision requests.
4. **Select & Develop** — Choose one option to expand into a full outcome: narration, a lasting consequence, next steps at the table, and safety/age notes.
5. **Output** — The GM uses this to describe the next story actions to the table.
